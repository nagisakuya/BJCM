ブラックジャックカウンティングマスター

BJCM(ブラックジャックカウンティングマスター)は、オンラインカジノでのブラックジャックのカウンティングをサポートするツールで、 カウンティングに基づいた正確なストラテジーとRTP(期待値)を算出することができます。

他の多くの類似ソフトは、あくまでも概算した結果を表示しているだけに過ぎません。なので正確なRTPとストラテジーからのズレが発生するため、細かい部分で損失が出てしまいます。BJCMは正確な計算を行い、正確なRTPとストラテジーを導出できるので利益を最大化することが出来ます。

使用方法

画面の説明


①ディーラーの手札です。
②プレイヤーの手札です。手札の下に計算された最適手が表示されます。左右の+-のボタンでプレイヤーを増やしたり減らしたりできます。また、カードを配る前に◆が表示されているのはあなたの手札で、勝敗が記録され、資金が増減します。これはクリックすることで切り替え出来ます。
③今のデッキの中身を示しています。概算RTPは通常のカードカウンティングを使用して計算される値を表示しています。
④再生ボタンを押すとRTPの計算が始まります。もう一度押すと止まります。また、bet:の部分に示されているのが、ケリー基準に基づいた推奨される掛け金です。この掛け金を掛けているものとして資金が増減します。

ご利用の前に

テーブルルールに合わせて、ルール設定からルールを変更してください!

正確なRTPのとストラテジーを計算するのに必ず必要です。正しくテーブルルールが設定されていないと正しい計算結果が出ません。

操作説明

数字キーを押すとカードを出せます。ディールの間は自動的にカードを出す場所が移動します。その後はスペースキーで次の人の手札に移動できます。エンターを押すと次のゲームになります。
上下左右キー:移動
S:選択中のハンドをスプリット
Z:一つ戻す
R:リセット(デッキの中身が初期状態に戻ります)

どのキーを使うかは、キー設定から自由に変更できます。

要求スペック

最低スペック 推奨スペック
CPU Intel i5 9600 Intel i7 12700
メモリ 16GB 32GB
OS Windows 10 Home Windows 10 Home
SSD 64GB 64GB

CPUの性能がRTPの計算速度に直結します。例えばIntel i5 9600では通常のブラックジャックの計算を約30秒に1回行えます。また、チャーリーの有無とデッキサイズが計算時間に大きな影響を与えます。
