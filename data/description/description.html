ブラックジャックカウンティングマスター

BJCM(ブラックジャックカウンティングマスター)は、オンラインカジノでのブラックジャックのカウンティングをサポートするツールで、 カウンティングに基づいた正確なストラテジーとRTP(期待値)を算出することができます。

他の多くの類似ソフトは、あくまでも概算した結果を表示しているだけに過ぎません。なので正確なRTPとストラテジーからのズレが発生するため、細かい部分で損失が出てしまいます。BJCMは正確な計算を行い、正確なRTPとストラテジーを導出できるので利益を最大化することが出来ます。

使用方法

画面の説明


①ディーラーの手札です。
②プレイヤーの手札です。手札の下に計算された最適手が表示されます。
③他の人の引いたカードを捨てる為の場所です。
④今のデッキの中身を示しています。
⑤再生ボタンを押すとRTPの計算が始まります。もう一度押すと止まります。

操作説明

上下左右キーでカードを出す場所を選択し、数字キーで選択された場所にカードを出します。絵札は0で出せます。次のゲームに進む時はエンターを押します。
S:スプリット
Z:一つ戻す
R:リセット

要求スペック

最低スペック 推奨スペック
CPU Intel i5 9600 Intel i7 12700
メモリ 16GB 32GB
OS Windows 10 Home Windows 10 Home
SSD 64GB 64GB

CPUの性能がRTPの計算速度に直結します。例えばIntel i5 9600では8デッキの計算を約1分に1回行えます。
